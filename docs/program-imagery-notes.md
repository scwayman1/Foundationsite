# Program Imagery Notes

## Current Principle

Program-area imagery should feel narratively specific, not decorative.

## Rules

- Avoid repeating campus/building shots across multiple program cards
- Prefer people, hands-on environments, and student-centered imagery
- If a photo does not clearly reinforce the category, remove or replace it
- Distinctiveness matters more than filling every slot with an image

## Next Cleanup Opportunity

If stronger category-specific visuals become available, update these first:
- Global Trade & Logistics
- Sustainability & Green Tech
- Advanced Manufacturing

Those three categories currently use the best available in-repo images, but they would benefit from more field-specific photography over time.
